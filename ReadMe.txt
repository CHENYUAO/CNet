|数据处理部分(Data.h&Data.c)：																					   |
|目前用10组数据进行测试，输入是一个10*11的二维数组																	   |
|文件定义了三个函数，分别是求均值、求标准差和数据处理。																   |
|																												   |
|求均值getAverage()和求标准差getStdDeviation()的使用需要注意，要先申请一段大小为sizeof(double)*DATA_DIMENSION			   |
|的内存，将指针作为函数的第二个参数传递进去，这样做的目的能够在函数外申请内存，函数外释放内存。							   |
|																												   |
|数据处理函数dataHandle()对二维数组的数据进行遍历，将空数据和异常数据填充为均值，并进行标准差标准化处理。					   |
|其中空数据这里暂时定义为DBL_MAX也就是double型的最大值，实际使用过程中还要根据数据采集模块进行修改。						   |
|																												   |
|――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――|																												   |
|网络部分(Net.h&Net.c):																							   |
|Net.h中主要是网络参数和测试数据，目前测试只能以数组形式先定义在头文件里，后续再进行优化									   |
|Net.c中是模型重构的实现，由于最终模型还没有确定下来，目前是用的2场景分类进行了测试。									   |
|需要注意的是矩阵运算函数的使用方式，在后续Matrix部分会详细解释													       |
|																												   |
|――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――――|
|矩阵部分（Matrix.h&Matrix.c）：																					   |
|定义了一些矩阵的运算，具体如下：																					   |
|initialize()：用于初始化矩阵的地址																				   |
|multiply()：矩阵乘法运算函数，使用前需要先申请存储结果的内存															   |
|add()：矩阵加法运算函数，使用前需要先申请存储结果的内存																   |
|show()：在控制台上显示矩阵																						   |
|sigmoid()：sigmoid激活函数，该函数只能运算一维向量																	   |
|relu()：relu激活函数，该函数只能运算一维向量																		   |
|maxIndex()：查找最大值并返回其次序（下标+1），该函数只能运算一维向量													   |
|addDim()：将一维向量长度扩展一位，最后一位置1，该函数只能运算一维向量													   |
|具体使用可以看Matrix.h文件的注释																					   |
